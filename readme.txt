== Description ==

Allows you to display a list of recent twitters  within a particular twitter user using your keys or any public feeds without keys.
follow this  link to learn more details
http://itfoundation-om.com/get-twitter-api-keys/




== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. now  just go to widget area and  use it
